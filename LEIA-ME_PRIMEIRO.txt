┌───────────────────────────────────────────────────────────────────────┐
│  🚀 COMO INICIAR O SISTEMA (PRIMEIRA VEZ)                            │
└───────────────────────────────────────────────────────────────────────┘

   1️⃣  Certifique-se que Node.js está instalado
       Download: https://nodejs.org/ (versão LTS)

   2️⃣  Duplo clique no arquivo:
       📄 INICIAR_SISTEMA.bat

   3️⃣  Aguarde as janelas abrirem (pode demorar 1-2 min na primeira vez)

   4️⃣  O navegador abrirá automaticamente em:
       🌐 http://localhost:5173

   ✅  PRONTO! O sistema está rodando!

┌───────────────────────────────────────────────────────────────────────┐
│  ⚙️  CONFIGURAÇÃO INICIAL (OBRIGATÓRIO)                              │
└───────────────────────────────────────────────────────────────────────┘

   Após o sistema abrir:

   1️⃣  Clique no ícone de engrenagem (⚙️) na barra lateral esquerda

   2️⃣  Na aba "Configuração Fácil", preencha:
       • Nome do seu apartamento
       • Endereço completo
       • URLs iCal do Airbnb e Booking

   3️⃣  Clique em "Salvar Configurações"

   4️⃣  Volte ao Dashboard (🏠) para ver as estatísticas

┌───────────────────────────────────────────────────────────────────────┐
│  🔗 ONDE OBTER AS URLs iCAL                                          │
└───────────────────────────────────────────────────────────────────────┘

   AIRBNB:
   1. Acesse: https://www.airbnb.com/hosting
   2. Vá em: Calendário → Disponibilidade
   3. Clique em: "Exportar calendário"
   4. Copie o link que aparece

   BOOKING.COM:
   1. Acesse: https://admin.booking.com/
   2. Vá em: Calendário → Sincronização
   3. Copie o link de exportação iCal

┌───────────────────────────────────────────────────────────────────────┐
│  🖥️  O QUE VAI ABRIR                                                 │
└───────────────────────────────────────────────────────────────────────┘

   Serão abertas 3 janelas:

   1. 🔵 SENTINEL - Backend API (terminal preto)
      → Não feche! É o servidor backend
      → Roda em: http://127.0.0.1:8000

   2. 🟢 SENTINEL - Frontend React (terminal preto)
      → Não feche! É o servidor frontend
      → Roda em: http://localhost:5173

   3. 🌐 Navegador (Chrome/Edge/Firefox)
      → É aqui que você vai usar o sistema!
      → Interface web completa

┌───────────────────────────────────────────────────────────────────────┐
│  🛑 COMO PARAR O SISTEMA                                             │
└───────────────────────────────────────────────────────────────────────┘

   Simplesmente FECHE as 2 janelas de terminal que foram abertas:
   • SENTINEL - Backend API
   • SENTINEL - Frontend React

   Ou pressione Ctrl+C em cada uma delas.
